# Sound assets

Place **camera-pop.mp3** in this folder for a short “pop” sound when capturing a photo.

**Description:** A short, cute popping sound like a camera shutter but softer (e.g. 0.2–0.5 seconds).

The app will play it when the user taps “Capture”; if the file is missing, it fails silently and the rest of the app still works.

You can use free sound effects from Freesound.org or similar (check license for use).
