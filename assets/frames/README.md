# Frame assets

Place these **transparent PNG** files in this folder. The app looks for:

| Filename           | Description |
|--------------------|-------------|
| `flower-frame.png` | Transparent PNG frame with pink flowers in the corners, soft watercolor style. |
| `star-sparkles.png`| Transparent PNG with gold star sparkles scattered around the edges. |
| `heart-corners.png`| Transparent PNG with cute red hearts in all four corners. |

**Size:** Same aspect ratio as the video (e.g. 4:3) or square; the app scales the frame to fit the capture.

You can create these in Canva, Figma, or use free resources (e.g. Flaticon, then add transparency). The repo may include placeholder or generated frames you can replace.
